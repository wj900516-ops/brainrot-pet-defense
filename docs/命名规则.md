# 命名规则参考表

## Workspace 结构
```
Workspace/
  Map/                    -- 地面、墙壁、装饰
  PathNodes/              -- 路径点文件夹
    Node1                 -- Part, 路径第1点
    Node2
    Node3
    Node4
    Node5
    Node6
  TowerSpots/             -- 塔位文件夹
    Spot1                 -- Part, 绿色 6x6
    Spot2
    Spot3
    Spot4
  SpawnPoint              -- Part, 怪物出生位置
  Base                    -- Part, 基地位置
```

## ReplicatedStorage 结构
```
ReplicatedStorage/
  Config/
    UnitConfig            -- ModuleScript, 塔数据
    EnemyConfig           -- ModuleScript, 怪数据
    WaveConfig            -- ModuleScript, 波次数据
  Remotes/
    PlaceTower            -- RemoteEvent (Day 2)
    StartWave             -- RemoteEvent (Day 2)
    UpdateUI              -- RemoteEvent (Day 2)
  Units/
    ToastDog              -- Model (以后放)
  Enemies/
    LagBlob               -- Model (以后放)
    StinkyBreadBoss       -- Model (以后放)
```

## ServerScriptService 结构
```
ServerScriptService/
  GameManager             -- Script, 主控
  WaveManager             -- ModuleScript, 波次管理
  EnemyService            -- ModuleScript, 敌人管理
  UnitService             -- ModuleScript, 塔管理
```

## StarterGui 结构
```
StarterGui/
  MainUI                  -- ScreenGui
    CoinsLabel            -- TextLabel, 左上角
    WaveLabel             -- TextLabel, 顶部居中
    BaseHpLabel           -- TextLabel, 右上角
    PlaceTowerBtn         -- TextButton, 底部居中
```

## 命名规范

| 类别      | 规范                  | 例子               |
|----------|----------------------|-------------------|
| 文件夹    | PascalCase           | PathNodes         |
| 脚本      | PascalCase           | GameManager       |
| Part 名   | PascalCase           | SpawnPoint, Node1 |
| 变量      | camelCase            | gameState         |
| 常量      | UPPER_SNAKE          | MAX_WAVES         |
| 函数      | camelCase            | spawnEnemy()      |
| Config键  | PascalCase           | ToastDog, LagBlob |
| Remote    | PascalCase           | PlaceTower        |
